'Test Sequence' is the text extraction from an app flow, it includes the text extraction from GUI and the actions between each page transactions. The Test Sequence is the text extraction from an app flow, it includes the text extraction from GUI and the actions between each page transactions. The paragraph followed by 'Initial structure' or 'Output structure' is the text extracted from GUI, and the action that cause the GUI changes are between the text extraction of the last GUI and new page starting with 'Output structure'.
The Output Decision indicates whether there's android non-crash bugs(inconsistency, a discrepancy, a logical error) in the given test sequence. UI logical bugs include: (1)Incorrect interaction: the app’s subsequent behavior deviates from expectations, such as clicking a button named “category” redirects to the home screen; (2) Functionality does not take effect: the user successfully interacts, but the function doesn’t work; (3) UI element does not react: components do not react to user inputs, such as no reaction after clicking a button; (4) Data loss: the data entered by the user is unexpectedly cleared; (5) Language related: language issues in the international version of the app, such as unexpected language changes.
