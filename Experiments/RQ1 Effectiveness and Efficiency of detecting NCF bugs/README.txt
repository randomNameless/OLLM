This section contains the experiment data from RQ1: Effectiveness and Efficiency of detecting NCF bugs.
Result table.xlsx contains the result of the experiment, including the bug information, and test result of TP rate(on sequence which contains bugs) and TN rate (on sequence which does not contain bugs) for OLLM-P1, OLLM-P2 and OLLM.
The folder Prompt of TP contains prompts to use for the TP experiments and the LLM response.
The folder Prompt of TN contains prompts to use for the TN experiments and the LLM response.
